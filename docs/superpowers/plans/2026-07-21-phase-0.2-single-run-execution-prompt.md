### Task 3: Generate Windows AppIds and Apply the Minimal Product Identity

**Files:**

**Interfaces:**

- Produces: valid Writing Buddy product identity and four canonical, non-conflicting Windows AppIds.

- [ ] **Step 1: Parse the original AppIds and preserve each exact wrapper**

Use this GUID pattern:

```powershell
$guidPattern = '[0-9A-Fa-f]{8}-[0-9A-Fa-f]{4}-[0-9A-Fa-f]{4}-[0-9A-Fa-f]{4}-[0-9A-Fa-f]{12}'
$appIdFields = @('win32x64AppId','win32arm64AppId','win32x64UserAppId','win32arm64UserAppId')
$beforeProduct = Get-Content -LiteralPath '.\product.json' -Raw | ConvertFrom-Json
$originalAppIds = [ordered]@{}
$appIdWrappers = [ordered]@{}
foreach ($field in $appIdFields) {
    $value = [string]$beforeProduct.$field
    $match = [regex]::Match($value, "^(?<prefix>.*?)(?<guid>$guidPattern)(?<suffix>.*?)$")
    if (-not $match.Success) { throw "Cannot parse original $field wrapper: $value" }
    $originalAppIds[$field] = $match.Groups['guid'].Value.ToUpperInvariant()
    $appIdWrappers[$field] = [pscustomobject]@{
        Prefix = $match.Groups['prefix'].Value
        Suffix = $match.Groups['suffix'].Value
    }
}
```

Do not assume whether the wrapper contains one or two braces. The captured prefix and suffix are authoritative.

- [ ] **Step 2: Generate four distinct uppercase GUID bodies and reconstruct the values**

```powershell
$generatedAppIds = [ordered]@{}
$generatedBodies = [System.Collections.Generic.List[string]]::new()
foreach ($field in $appIdFields) {
    do {
        $body = [guid]::NewGuid().ToString().ToUpperInvariant()
    } while ($originalAppIds.Values -contains $body -or $generatedBodies.Contains($body))
    $generatedBodies.Add($body)
    $generatedAppIds[$field] = $appIdWrappers[$field].Prefix + $body + $appIdWrappers[$field].Suffix
}
$generatedAppIds | ConvertTo-Json | Out-File -LiteralPath (Join-Path $runtimeRoot 'logs\generated-appids.json') -Encoding utf8
```

Expected: four values whose GUID bodies are valid, pairwise distinct, absent from all original AppIds, and whose wrappers exactly match the corresponding original fields.
